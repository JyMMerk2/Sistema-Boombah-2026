Manual Production Control

VERSIÓN 2026

Hecho por: Juan Mercado | Planta: BOOMBAH DOMINICANA

1. Digitación de Reportes

Identifique el tipo de cuadre antes de iniciar:

  • Cuadre F: Para órdenes de Full Dye (Apparel).
  • Cuadre M: Para órdenes de Mochilas (Bags).

Importante: El PO debe coincidir exactamente con el estilo reportado en la hoja del supervisor.

2. Validación en NewSoft

Al terminar, el total del reporte físico debe ser idéntico al digital.

En caso de error de estilo:
  1. Valide el PO en NewSoft o NetSuite.
  2. Si es Mochila, el supervisor debe validar su historial manual si el PO no es legible.

3. Captura de Producción

Pasos para la captura diaria:

  • Programa: Use CTRL + I para insertar el programa del día.
  • Subprocesos: Apparel (14) / Bags (13).
  • Guardar: Use la tecla F9 para salvar la captura.
